# PUBLIC ENEMY 맞추기

유튜브 구간을 짧게 듣고 세 곡 중 정답을 맞히는 모바일 우선 웹게임입니다.

## 플레이 방식

- `듣기`를 누르면 현재 라운드의 고정 구간이 재생됩니다.
- 다시 들을수록 재생 길이가 늘어납니다.
- 정답을 맞히면 점수를 얻고 다음 라운드로 넘어갑니다.
- 틀리면 현재 런이 종료되고 10초 뒤 새로 시작합니다.

## 점수

- 첫 청취 정답: `100`
- 1번 연장 후 정답: `50`
- 2번 연장 후 정답: `10`
- 3번 연장 후 정답: `0`
- 4번 연장 후 정답: `-50`

최고 기록은 브라우저 로컬 저장소에 저장됩니다.
